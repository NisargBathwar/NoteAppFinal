# Notes App

A simple Notes application built using Jetpack Compose and MVVM architecture.  
This app allows users to create, edit, delete, and manage notes with a clean UI and proper state management.

## Features
- Add new notes
- Edit existing notes
- Delete notes
- Restore deleted notes
- StateFlow based UI updates
- MVVM architecture

## Tech Used
- Kotlin
- Jetpack Compose
- ViewModel
- StateFlow
- MVVM Architecture
- Repository Pattern

## Architecture
UI → ViewModel → Repo → State → UI

This project was built to practice clean architecture and state management in Android.
