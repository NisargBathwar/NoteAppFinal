📝 NoteApp — Google Keep Style Notes App
A feature-rich Notes application built with Jetpack Compose and Clean MVVM Architecture.
Inspired by Google Keep, this app goes beyond basic CRUD with gestures, filtering, theming, and smart search.

📱 Features
🗒️ Note Management

✅ Create, Edit, and Delete notes
✅ Color-coded notes (Google Keep style)
✅ Pin / Unpin notes for quick access
✅ Soft delete with Snackbar 

👆 Gesture Support

✅ Swipe Left → Delete note
✅ Swipe Right → Pin / Unpin note

🔍 Search & Filter

✅ Search notes by title/content with Debounce (performance optimized)
✅ Filter by Pinned / Unpinned notes
✅ Sort by Newest First / Oldest First

🎨 UI / UX

✅ Toggle between Grid and List layout
✅ Clean, minimal UI with smooth state transitions
